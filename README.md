# Basic of Android Networking
Basic networking implementation in Android (Ktor and/ or Retrofit)

A simple application that manage Movie catalog, this implements CRUD the four fundamental operations used to manage and manipulate data.

Using this [RESTful API](https://github.com/neilmosca25/basic-rest-api-express-mongodb), I create an android application that utilizes usage HTTP Client libraries such as Retrofit and/ or Ktor.
